The goal of this project is to practice building REST APIs using FastAPI while learning backend development concepts. As I continue learning, I'll keep adding new features and improving the project.

- Query Parameters

- Pydantic Models
- CRUD Operations
- Response Models
- Error Handling
- File Structure
- More features coming soon...

- FastAPI

- Pydantic

## Run the Project

1. Clone the repository

```bash
git clone https://github.com/<your-username>/fastapi-demo-project.git
```

2. Create a virtual environment

```bash
python -m venv .venv
```

3. Activate it

**Windows**

```bash
.venv\Scripts\activate
```

**Linux/macOS**

```bash
source .venv/bin/activate
```

4. Install dependencies

```bash
pip install -r requirements.txt
```

5. Start the server

```bash
uvicorn main:app --reload
```

6. Open the API documentation

- Swagger UI: http://127.0.0.1:8000/docs
- ReDoc: http://127.0.0.1:8000/redoc
